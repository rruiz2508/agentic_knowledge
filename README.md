# Agentic GPT Knowledge Base

Base de conocimiento inicial para el GPT personalizado **Agentic Software Architect**.

## Objetivo

Este conjunto de archivos sirve como conocimiento base para un GPT especializado en programación agéntica, diseño de agentes IA, documentación de skills, LangChain, LangGraph, prompts, herramientas, memoria, seguridad y buenas prácticas de ingeniería.

## Archivos incluidos

| Archivo | Propósito |
|---|---|
| `01_user_context.md` | Contexto del usuario, stack técnico, preferencias y forma de trabajo. |
| `02_agent_design_standard.md` | Plantilla estándar para diseñar y documentar agentes. |
| `03_skill_template.md` | Plantilla para crear y documentar skills reutilizables. |
| `04_prompt_engineering_standard.md` | Estándar para diseñar prompts robustos y mantenibles. |
| `05_langgraph_reference.md` | Guía práctica para diseñar workflows y agentes con LangGraph. |
| `06_langchain_reference.md` | Guía práctica para usar LangChain en soluciones agénticas. |
| `07_dotnet_agentic_guidelines.md` | Recomendaciones para integrar agentes IA en proyectos .NET. |
| `08_security_guardrails.md` | Guardrails de seguridad para agentes, tools y automatizaciones. |
| `09_documentation_templates.md` | Plantillas reutilizables de documentación técnica. |
| `10_tooling_stack.md` | Mapa de herramientas: ChatGPT, Copilot, Roo Code, DeepSeek, Gemini y VS Code. |

## Forma recomendada de uso

1. Subir estos archivos al apartado **Knowledge** del GPT personalizado.
2. Mantenerlos versionados en GitHub o Google Drive.
3. Actualizarlos cada vez que se definan nuevos estándares de trabajo.
4. Usar este repositorio como base para prompts, agentes, workflows y documentación técnica.

## Principio rector

ChatGPT y el GPT personalizado actúan como consultores estratégicos y arquitectos.  
Herramientas como GitHub Copilot, Roo Code, DeepSeek y otros agentes actúan como constructores o ejecutores.
